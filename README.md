# fork from [https://github.com/lstprjct/IDM-Activation-Script](https://github.com/lstprjct/IDM-Activation-Script)
 
- 打开power shell
- 输入 `IWR -USEB https://raw.githubusercontent.com/lstprjct/IDM-Activation-Script/main/IAS.ps1 |IEX`
- 自定跳转到cmd，输入“1”，然后在输入“9”，等待下载完成，即可激活成功

**意：IWR -USEB 必须大写；**
